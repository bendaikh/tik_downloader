üîß Installation Guide

Prerequisites

Before installing, ensure your server meets these requirements:

Server Requirements

  • PHP Version: 8.0.2 or higher (8.1.0+ recommended)
  • Web Server: Apache or Nginx
  • Database: MySQL 5.7+ or MariaDB 10.2+

PHP Extensions

  • cURL (7.55.0+)
  • OpenSSL (1.0.2k+)
  • PDO
  • fileinfo
  • mbstring
  • xml
  • json
  • zip
  • gd
  • intl

PHP Settings

  • allow_url_fopen = On
  • memory_limit = 128M or higher
  • max_execution_time = 300 or higher
  • upload_max_filesize = 64M or higher
  • post_max_size = 64M or higher

Installation Steps

  1. Upload Files

    Download the script from CodeCanyon, extract the ZIP file, and upload all files to your web server's public directory (usually public_html or www). Make sure the files are uploaded to the root of your domain.

  2. Set Permissions

    Set the following directory permissions:

    chmod 755 artisan chmod 644 public/index.php chmod -R 755 storage/ chmod -R 755 bootstrap/cache/ chmod 644 themes/config.json

  3. Run Installation Wizard

    Open your browser and navigate to: https://yourdomain.com/install. The installation wizard will automatically start and guide you through the setup process.

Installation Wizard Steps

  1. Introduction

    Read and accept the terms of service, then click "Agree and Start Installation".

  2. Requirements Check

    The system will automatically check all server requirements. Make sure all requirements are met (green checkmarks) and click "Continue".

  3. Database Configuration

    Enter your database details and click "Migrate" to set up the database automatically.

  4. Create Admin Account

    Enter your admin account details and click "Create Admin" to create your account.

  5. Finish Installation

    Click "Clean Up and Finish Installation" to complete the setup.

üìñ Usage Guide

How to Download TikTok Videos

  1. Copy TikTok URL

    Open TikTok and copy the URL of the video you want to download.

  2. Paste URL

    Paste the TikTok URL into the input field on the homepage.

  3. Choose Format

    Select your preferred format (Video or Audio).

  4. Download

    Click the "Download" button and wait for processing to complete.

Supported URL Formats

https://www.tiktok.com/@username/video/1234567890123456789 https://vm.tiktok.com/XXXXXXXX/ https://m.tiktok.com/v/1234567890123456789.html

Download Options

Format Description Quality
Video Download video without watermark Original quality
Audio Extract audio from video MP3 format

Tips for Best Results

  • Make sure the TikTok video is public and accessible
  • Use the official TikTok app URL when possible
  • Check your internet connection for faster downloads
  • Some videos may be restricted by region or privacy settings

Common Issues & Solutions

Installation Issues

Database connection failed
  • Check your database credentials
  • Ensure the database exists
  • Verify database user permissions

  • Set correct file permissions
  • Ensure web server can write to storage directory
  • Check SELinux settings (if applicable)

Download Issues

  • Check if the video is public and accessible
  • Verify the TikTok URL is correct
  • Try refreshing the page and retrying
  • Some videos may be region-restricted

Performance Issues

  • Check your internet connection
  • Clear browser cache and cookies
  • Try using a different browser
  • Contact your hosting provider if issues persist

üîå API Reference

RESTful API Endpoints

The TikTok Video Downloader includes a comprehensive API for third-party integrations.

Download Video

POST /api/download Content-Type: application/json { "url": "https://www.tiktok.com/@username/video/1234567890123456789", "format": "video" } Response: { "success": true, "data": { "download_url": "https://yourdomain.com/download/abc123", "title": "Video Title", "duration": "00:30", "size": "5.2MB" } }

Get Video Info

GET /api/info?url=https://www.tiktok.com/@username/video/1234567890123456789 Response: { "success": true, "data": { "title": "Video Title", "author": "@username", "duration": "00:30", "views": "1000", "likes": "500" } }

Authentication

API requests require authentication using API keys. Add the following header to your requests:

Authorization: Bearer YOUR_API_KEY

Rate Limits

  • Free Tier: 100 requests per hour
  • Premium Tier: 1000 requests per hour
  • Enterprise: Custom limits available

Error Codes

Code Description
